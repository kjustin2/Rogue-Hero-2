# Rogue-Hero-2

Tempo-driven roguelike deck builder. HTML5 Canvas + vanilla JS — no build step, no dependencies. Single-player, local 2-player co-op, or up to 4-player remote co-op over WebRTC.

The project itself lives in [`Rogue-Hero-2/`](./Rogue-Hero-2/) — see that directory's `README.md` for how to play and how to package.
